# Verifica TypeScript Base

## Obiettivo
Completare la verifica modificando solo i file consentiti.

## File che PUOI modificare
- quiz/teoria.md
- src/types.ts
- src/tasks/01_registro.ts
- src/tasks/02_statistiche.ts
- src/tasks/03_async.ts
- src/debug/debug1.ts
- src/debug/debug2.js

## File che NON devi modificare
- README.md
- package.json
- tsconfig.json
- src/main.ts
- src/data.ts

## Istruzioni tecniche
1. Apri la cartella in Visual Studio Code.
2. Apri il terminale nella cartella principale.
3. Esegui:
   npm install
4. Durante il lavoro puoi controllare il codice con:
   npm run check
5. Per compilare:
   npm run build
6. Per eseguire il programma principale:
   npm run start

## Attenzione
- Non cambiare i nomi dei file.
- Non creare nuove cartelle.
- Non spostare i file.
- Non eliminare codice già presente se non serve a correggere un errore.
- Nei file di debugging correggi il codice senza riscriverlo da zero.
- Nei file di debugging aggiungi un breve commento sopra ogni correzione fatta.

## Consegna finale
Quando hai finito:
1. controlla che i file siano salvati;
2. comprimi l'intera cartella della verifica in un file zip;
3. rinomina il file zip in questo modo:
   nome-cognome.zip
4. carica il file zip su Google Classroom.
